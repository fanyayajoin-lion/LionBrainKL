## 摘要

- Narrator AI CLI 是一套可透過指令列自動生成電影旁白影片的 AI 工具
- 搭配 SKILL.md 可讓 AI 代理（OpenClaw、Windsurf、Claude Code 等）直接理解並執行影片製作流程
- 工作流程：搜尋電影 → 選模板 → 配 BGM → 選聲優 → 生成腳本 → 組裝影片 → 回傳下載連結
- 安裝方式：pip install narrator-ai-cli，需配置 API key（merlinyang@gridltd.com）
- CLI 是執行工具，Skill 是腦——兩者搭配讓 AI 能從頭到尾自主完成影片製作
